# Vigil 0.1.0-beta.1

First public beta of Vigil — a macOS menu-bar app and CLI for keeping a Mac fully awake while the lid is closed.

This is a **prerelease**. Sparkle's stable channel (`releases/latest/download/appcast.xml`) does not include it, so existing installs (once any exist) will not auto-update onto it.

## What works
- Reversible `pmset` awake profile (`disablesleep`, `sleep`, `disksleep`, `ttyskeepawake`, `tcpkeepalive`).
- Persistent user LaunchAgent that takes IOKit power assertions only while enabled.
- Lid-state watcher that dims displays + keyboard backlight on close and restores on open.
- One-time scoped sudoers approval so Enable / Disable run without repeated password prompts.
- Sparkle 2 auto-update from GitHub Releases (EdDSA-signed appcast).

## Install
1. Download `Vigil-0.1.0-beta.1.zip` from this release page.
2. Unzip it.
3. **Drag `Vigil.app` to `/Applications` before opening it.** Launching from `~/Downloads` triggers macOS App Translocation; Vigil refuses to enable in that state.
4. Open `Vigil.app`. The menu bar icon appears at the right of the status bar.

## Safety
Closed-lid operation can trap heat. Keep the machine ventilated and prefer AC power. Vigil refuses to enable on battery power unless you click "Enable on Battery".

## Known limitations of this beta
- The first stable `v0.1.0` release will be the one Sparkle's `latest` channel serves.
- Auto-update testing requires shipping the eventual stable `v0.1.0` after this beta validates the release pipeline end-to-end.
